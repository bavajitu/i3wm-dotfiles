## i3wm based Arch Linux workflow:

I don't like a very fancy, good looking, "RICED" desktop, even though I used to have one like that myself. I just want my workflow to be as fast as it can be, while being able to make things for me a little easier.
These are the things I always expect my workflow to have:

- A very low, very very low idle RAM usage.
- No unused bloat, everything serves a purpose.
- Non-distracting (I don't want to be staring at my waifu wallpaper through my transparent pretty terminal while working on a seg fault).
- No unwanted startup software, unless I want it to be.
- Should be easy to replicate on a new system.

**Note**: These are only the base requirements, this is the philosophy I work with. It's arbitrary, every single one of them.
